# Privacy Policy

Stb_Chatbot collects only the information needed to provide the chatbot service.

## Data Collected

- Account name and email
- OAuth account identifiers from Google/GitHub
- Chat messages and conversation history
- Uploaded documents/files used as chat context
- Usage data such as message count and token count
- IP/request metadata in server logs

## Purpose

Data is used to:

- Authenticate users
- Provide chat history
- Enforce daily limits
- Process uploaded files and GitHub repository context
- Improve reliability and prevent abuse

## Uploaded Files

Uploaded files may be stored on the server under `backend/uploads`. Users should avoid uploading sensitive credentials, secrets, private keys, passwords, or confidential documents.

## Data Sharing

User data is not sold. Data may be sent to AI providers such as X5Lab to generate responses.

## Data Deletion

Users can delete their account from Settings. Deleting an account removes the account and related chat history from the database.
